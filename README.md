# Flight Price Prediction

This project focuses on predicting flight ticket prices using machine learning techniques and analyzing key factors affecting pricing.

## Project Overview

The objective is to build a predictive model capable of estimating flight prices based on features such as airline, route, number of stops, and ticket class.

## Technologies Used

- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn
- Power BI

## Project Structure

- `notebook/`: Data analysis and model development
- `data/`: Dataset used for training
- `dashboard/`: Power BI dashboard

## Key Insights

- Ticket class significantly impacts price
- Flights with fewer stops tend to be more expensive
- Airline and route influence pricing
- Price varies over time

## Models Used

- Linear Regression
- ElasticNet

## Next Steps

- Try tree-based models (Random Forest, XGBoost)
- Improve generalization
- Hyperparameter tuning
- Deploy as an API or dashboard
